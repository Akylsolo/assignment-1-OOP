# Student Management System

## Overview
This project demonstrates core Object-Oriented Programming concepts in Java,
including encapsulation, classes, objects, arrays, and composition.

## Classes
### Student
Stores student information such as name, ID, major, GPA, and credits.
Includes methods to update GPA, add credits, and check honors status.

### Course
Represents a course containing an array of students.
Calculates average GPA and finds the student with highest credits.
